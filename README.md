# Portfolio Project

This repository is a personal workspace for building and experimenting with a portfolio-style website.  
It is not intended as a finished product — instead, it serves as a testing ground for UI ideas, layouts, and Git workflows.



## 🧠 Purpose

- Experiment with frontend concepts
- Practice structuring a website project
- Test different UI/UX approaches
- Improve Git and branching workflow
- Deploy and verify changes in a live environment



## 🚧 Project Status

Ongoing — features and structure may change frequently.  
This repo is used for iteration, not stability.



## 🔁 Workflow

- Development happens on the `dev` branch  
- Changes are tested before merging into `main`  
- Unused branches are removed to keep things clean  



## 🌐 Deployment

Deployed using Vercel for quick previews and testing.



## ⚠️ Notes

- This is a personal project, not production-grade software  
- Code may be refactored, replaced, or removed at any time  
- Focus is on learning and experimentation, not perfection
